Pseudocode as comments:

Task 1: Pseudocode an everyday task -- Doing the dishes

  1. Gather the dirty dishes.
  2. Scrape any food on them into the trash.
    • Use fork to scrape food away
  3. Put the dishes in the sink.
  4. Grab the sponge and apply soap and water.
  5. Use the sponge in circular motion to remove grime.
    • If food is caked on, scrub hard with hot water
  6. Rinse off and set dishes on a flat surface to dry.

Task 2: Pseudocode a simple computer game -- Rock Paper Scissors

  1. Give the first user 3 options to pick from: Rock, Paper, or Scissors.
    • Save this answer
  2. Give the second user the same 3 options to pick from.
    • Save this answer
  3. Compare the two saved answers against the stored rule conditions
    • If tie, repeat.
    • If there is a winner announce the user with the winning task.
